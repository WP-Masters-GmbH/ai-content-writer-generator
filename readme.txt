= What's inside? =
* AI Articles Generation by given topics
* One AI Article Per Day with "FREE" license code
* Extra Premium Packages available with no daily limits
* One-Click Posts Import

= Popular use cases =
* You need to write articles but don't know where to get ideas and start from
* You need to quickly get a lot of content

= How to use it =
* Install and Activate AI Writer & Generator plugin
* Use "FREE" licence code or buy more articles
* Put a list of topics to get ideas for, select posts category for articles to be added in
* Click "Send to Assistant" and wait for ideas to be created
* Where content ideas are ready - click "Preview Content" or "Import"
